Njord Project

A cryptocurrency algorithmic trading project powered by machine learning.

Learn more

Machine Learning

We collect, store and manipulate data exctracted from multiple online exchanges to feed different machine learning models. The output are used to power our trading strategies.

Supervised

Supervised Learning models are developed to make predictions about the future price of the different cryptocurrencies. We developed Dense, Recurrent, LSTM and Convolutionnal Neural Network.

Reinforcement

DQN

Deep Q-network

DDPG

Deep Deterministic Policy Gradient

RRL

Recurrent Reinforcement Learning

Open Source

Njord Project is an open-source project available on github. There you will get a simplification of our pipe in order to assess the power of the tools we created. If you want more data, more complex machine learning models, more information, please contact us.
